# Zplit-testwebsite


To run this locally :-

1. Clone the repository:
   ```bash
   git clone https://github.com/Purnavu-12/Zplit-testwebsite.git
    cd Zplit-testwebsite
    ```
2. npm install
   ```bash 
    npm install -g http-server
   ``` 
3. Start the server:
   ```bash
    http-server
   ```
4. Open your web browser and navigate to `http://localhost:8080` (or the port specified in the terminal) to view the website.

This is a test website for Zplit.
